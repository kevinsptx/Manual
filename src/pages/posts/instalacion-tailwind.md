# Instalación de Tailwind CSS

Tailwind CSS es una herramienta que permite diseñar paginas web usando clases ya preparadas.

Pa instalar Tailwind en Astro utilicé el siguiente comando:

```bash
npx astro add tailwind
```

Durante la instalacion aparecieron algunas preguntas y simplemente seleccioné la opcion de confirmar.

Despues de eso ya podia usar clases de Tailwind directamente.

Por ejemplo:

```html
<h1 class="text-4xl font-bold text-blue-500">
Hola mundo
</h1>
```

Con unas pocas clases ya podemos cambiar tamaños, colores y posiciones sin necesidad de escribir mucho CSS.

La verdad me pareció bastante comodo porque ahorra bastante tiempo.

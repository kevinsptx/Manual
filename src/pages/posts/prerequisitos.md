# Prerrequisitos

Antes de empezar con Astro hay algunas cosas que necesitamos tener listas en nuestro computador.

Lo primero es contar con una distribucion Linux instalada. En mi caso utilicé Ubuntu, pero tambien funciona en otras distribuciones.

Tambien necesitamos acceso a la terminal ya que la mayor parte de la instalacion se realiza desde ahi.

Es importante tener conexion a internet porque se van a descargar varios paquetes durante el proceso.

Pa verificar si ya tenemos Node.js y npm instalados podemos usar los siguientes comandos:

```bash
node -v
npm -v
```

Si ambos muestran una version significa que ya estan instalados.

Si alguno no funciona no pasa nada, mas adelante veremos como instalarlo.

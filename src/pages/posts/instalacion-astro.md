# Instalación de Astro

Una vez instalado Node.js ya podemos crear nuestro proyecto de Astro.

Pa crear un proyecto nuevo utilicé:

```bash
npm create astro@latest
```

El asistente hace varias preguntas sobre el nombre del proyecto y otras configuraciones.

Despues entré a la carpeta creada:

```bash
cd manual
```

Y ejecuté el proyecto:

```bash
npm run dev
```

Si todo sale bien aparece una direccion parecida a esta:

```bash
http://localhost:4321
```

Al abrirla en el navegador ya podemos ver el proyecto funcionando.

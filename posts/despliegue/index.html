Despliegue

Cuando terminamos el proyecto podemos generar la version final lista pa publicar.

El comando utilizado es:

npm run build

Esto crea una carpeta con todos los archivos optimizados.

Pa probar la version final usamos:

npm run preview

Si todo funciona correctamente el proyecto ya estaria listo pa subirlo a internet.

Por ejemplo se podria publicar en GitHub Pages, Netlify o Vercel.

Llegar hasta este punto fue interesante porque ya no era solo codigo, sino una pagina completa lista pa compartir.
